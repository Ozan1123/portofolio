👋 Tentang Saya

Muhamad Fauzan Pratama adalah calon programmer muda dari SMK Plus Pelita Nusantara. ...

Saya adalah siswa aktif di SMK Plus Pelita Nusantara, dengan fokus minat pada bidang rekayasa perangkat lunak, khususnya pengembangan web dan aplikasi.

Sejak awal mengenal dunia pemrograman, saya telah membekali diri dengan dasar-dasar penting seperti HTML, CSS, JavaScript, dan PHP, serta mulai mengembangkan proyek-proyek sederhana yang membantu saya memahami alur kerja dunia nyata. Beberapa proyek yang pernah saya kerjakan antara lain adalah aplikasi kalkulator online, website portofolio pribadi, dan kartu profil interaktif yang saya bangun menggunakan teknologi front-end.

Saya dikenal sebagai pribadi yang kritis dalam memecahkan masalah, senang belajar hal baru, dan mampu bekerja secara mandiri maupun dalam tim. Kemampuan problem solving, berpikir terstruktur, serta rasa ingin tahu yang besar membuat saya terus tertantang untuk mengembangkan diri lebih dalam, tidak hanya di sisi tampilan (frontend), tetapi juga logika program (backend).

Tujuan saya ke depan adalah menjadi seorang Full-Stack Web Developer yang mampu membangun aplikasi yang efisien, menarik secara visual, dan fungsional. Saya juga mulai tertarik dengan pengembangan mobile apps dan teknologi modern seperti framework JavaScript serta pengelolaan database.
